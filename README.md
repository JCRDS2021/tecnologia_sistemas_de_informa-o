# Tecnologia, Sistemas de Informação e Comunicação, Segurança Cibernética e Crimes Digitais.

OBJETIVO: Este repositório possui o link para um Notebook Lm, voltado para o estudo de Tecnologia, Sistemas de Informação e Comunicação, Segurança Cibernética e Crimes Digitais. Ele foi criado com o intuito de servir como um guia de estudos para o concurso da Polícia Civil do Paraná.

FONTES: As fontes que compõem o notebook trazem uma gama de conteúdos voltados para a área da Cibersegurança, bem como para o estudo sobre crimes digitais. O intuito deste Notebook é tirar dúvidas sobre os principais temas que serão cobrados no concurso, bem como para estudar assuntos relacionados à segurança cibernética, como por exemplo, a utilização de sistemas operacionais específicos na área, como o Linux, entre outras ferramentas necessárias para atuar no ramo.
